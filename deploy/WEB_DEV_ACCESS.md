# Web 版开发环境访问信息

## ✅ Next.js 开发服务器已启动！

### 本地访问地址

```
http://localhost:3000
```

### 局域网访问地址（可在手机/平板测试）

```
http://192.168.0.109:3000
```

### 服务状态

- ✅ 开发服务器运行中
- ✅ 热更新已启用
- ✅ 字体配置已临时禁用（使用系统默认字体）

### 使用说明

1. **本地电脑访问**：

   - 在浏览器打开 `http://localhost:3000`

2. **同一局域网内的其他设备访问**：

   - 确保设备与开发机在同一 WiFi 网络
   - 在设备浏览器打开 `http://192.168.0.109:3000`

3. **登录信息**：
   - 系统会重定向到登录页面
   - 使用现有的账号密码登录

### 后端 API 配置

当前 Web 应用连接的后端 API：

- 开发环境可能使用 `http://localhost:4000` 或其他配置
- 需要确保后端服务也在运行

### 停止服务

如需停止开发服务器：

```bash
# 查找进程
lsof -ti:3000

# 终止进程
kill $(lsof -ti:3000)
```

### 注意事项

1. ⚠️ 阿里巴巴普惠体字体暂时禁用，使用系统默认字体
2. ⚠️ 如需完整字体支持，需要下载字体文件并放置于 `web/public/fonts/` 目录
3. 开发服务器支持热更新，修改代码会自动刷新页面
